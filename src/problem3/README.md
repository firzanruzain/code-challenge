# Problem 3 – React Refactoring & Code Review

This folder contains the code and documentation for a React refactoring/code review challenge.

## Contents

- [messyReact.tsx](messyReact.tsx) – The original, unrefactored React component with code smells and anti-patterns.
- [refactoredReact.tsx](refactoredReact.tsx) – The improved, refactored version of the same component, applying best practices for readability, maintainability, and performance.
- [code-review.md](code-review.md) – A written review and summary of the refactoring process, highlighting issues found in the original code and the improvements made.
